Данный проект показывает работу с развлекательным приложением.

Команда разработчиков рекомендательных систем создала новый алгоритм рекомендаций, который, по их мнению, будет показывать более интересный контент для каждого пользователя. 
Была поставлена задача рассчитать параметры A/B-теста, который позволит проверить эту гипотезу, и проанализировать его результаты.
